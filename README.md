# GERMAN_LAB_MINI

## Проект Олега Германа

Этот каталог представляет собой "облегченную" версию проекта измерительной лаборатории, созданную для платформы MOKO SE по автоматизации измерений.

### Содержимое

*   **German's Measurement Mini**: Директория с результатами измерений (`.mpr`), сгруппированными по датам.
*   **data**: Содержит данные, включая `Results.csv` и изображения, относящиеся к тестовой структуре.
*   **Python-скрипты**: Набор скриптов с суффиксом `(mini)`, автоматизирующих процесс измерений:
    *   `1 Настройка параметров (mini).py`: Настройка параметров для "mini"-версии.
    *   `2 Инициализация приборов (mini).py`: Инициализация оборудования.
    *   `3 Измерения (mini).py`: Выполнение "mini"-последовательности измерений.
    *   `4 Создание отчета (mini).py`: Генерация отчета.
*   **Библиотечные файлы**: Модули `MFRT.py`, `MGPH.py`, `MOKO.py`, `MOSC.py`, предоставляющие вспомогательные функции.
*   **German's Measurement Mini.msep**: Файл проекта MOKO SE для "mini"-версии.
*   **Отчет.docx**: Шаблон или пример отчета.

### Назначение

Проект `GERMAN_LAB_MINI` является упрощенной или сокращенной версией измерительной лаборатории, предназначенной для выполнения базовых или быстрых тестов на платформе MOKO SE.
